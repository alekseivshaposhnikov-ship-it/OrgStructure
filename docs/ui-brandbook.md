5. Принципы интерфейса
Интерфейс следует фирменной айдентике компании, но сохраняет привычные UX-паттерны корпоративных информационных систем.
Основной фирменный цвет — Oceanic Blue (#213FDC).
Orange (#F75B19) используется только для акцентных действий и важных элементов интерфейса.
Не рекомендуется одновременно использовать более двух фирменных цветов на одном экране.
Все тексты интерфейса используют TT Norms Pro. Шрифт Oceanic применяется только в крупных заголовках и презентационных элементах.
Все тексты выравниваются по левому краю, кроме декоративных заголовков.
Главные действия должны визуально доминировать над вторичными.
Сервисные функции не должны конкурировать с основными пользовательскими сценариями.
Сценарное моделирование по умолчанию остается свернутым.
Экспорт PDF должен полностью сохранять структуру экранной диаграммы.
Левое дерево отображает только подразделения и не перегружается дополнительной информацией.
Документация изменений оформляется кратко: версия, дата, список изменений.
